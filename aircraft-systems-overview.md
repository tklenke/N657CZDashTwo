# Systems Overview

Major systems installed on N657CZ, with references to their sections.

---

## Flight Controls — Section 12

| System | Description |
|--------|-------------|
| Ailerons | Roll control, actuated by side stick. |
| Canard and elevators | Pitch control via full-span elevators on the canard. Electric pitch trim. |
| Rudders | Yaw control via winglet-mounted rudders, actuated by rudder pedals. |
| Landing brake | Electrically actuated drag device mounted on the lower fuselage. |

See Section 12 for inspection and adjustment procedures.

## Avionics — Section 17

All Garmin avionics. The majority of LRUs are located in the avionics bay forward of the IP. The GEA 24, GMU 11, and autopilot servos are located elsewhere — see [[General Layout and Configuration|aircraft-general-layout]] in Section 3.

| LRU | Function |
|-----|----------|
| GDU 460 | Primary flight display (PFD). |
| G5 | Backup flight instrument. |
| GNC 355 | GPS navigator with VHF comm. |
| GTR 20 | VHF communication radio. |
| GTX 45R | Transponder and ADS-B (in/out). |
| GMA 245R | Audio panel. |
| GSU 25C | ADAHRS (air data, attitude, heading reference). |
| GMU 11 | Magnetometer, right strake. |
| GEA 24 | Engine data acquisition, forward firewall. |
| GMC 507 | Autopilot controller. |
| GSA 28 (×2) | Autopilot servos — pitch (right fuselage, forward of IP) and roll (under cowling on strake). |
| GSB 15 | USB Power plug. |
| GPS 20A | GPS Source. |
| GA 35 | GPS antenna. |

See Section 17 for avionics inspection and maintenance.

The aircraft uses embedded copper foil antennas laminated into the composite structure (wings, canard, vertical stabilizers, fuselage) plus an external GPS antenna. See [[Antenna System|avionics-antennas]] in Section 17 for the full inventory.
